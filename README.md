Real-time social media pipeline processing 39,000+ events through

and AWS S3 + Athena cloud analytics.

| Layer | Tool | Purpose |
|---|---|---|

## 🧩 Core Expertise

### 🔄 Data Pipeline Engineering
- Medallion Architecture (Bronze / Silver / Gold)
- Batch ingestion with Hive-style date partitioning
- Pipeline orchestration with retries and structured logging
- Data quality gates — automated checks at every layer

### 🌊 Streaming & Real-Time
- Apache Kafka — topics, producers, consumers, offsets
- Consumer groups and partition-based scaling
- Windowed aggregations — sliding and tumbling windows
- Watermark handling for late-arriving data

### 🔧 Transformation & Modeling
- dbt Core — SQL models, sources, refs, lineage graphs
- dbt testing — unique, not_null, accepted_values, relationships
- Window functions, CTEs, deduplication patterns
- Star schema and RFM customer segmentation

### ☁️ Cloud & Storage
- AWS S3 — data lake storage, Hive partitioning
- AWS Athena — serverless SQL on S3 Parquet files
- AWS IAM — user management and permissions
- boto3 — Python SDK for AWS services

### 📊 Analytics & Visualization
- DuckDB — in-process SQL on Parquet/CSV/JSON
- Metabase — business KPI dashboards
- Chart.js — custom HTML dashboards
- SQL analytics — aggregations, time-series, segmentation
